Elemente Semantisch? Bedeutung
a yes Verweis (Hyperlink) und Anker für Verweise
abbr yes Abkürzung
acronym yes Akronym
adress yes Kontaktangabe zum Autor eines Inhalts (Credits)
area yes Verweis-sensitiver Bereich in einer Image-Map-Grafik
b no Stilistische Hervorhebung im Text (üblicherweise fett dargestellt)
base yes Basis-URL für Referenzen
bdo yes Richtungsänderung bei bidirektionalem Text
big no Großschrift
blockquote yes Zitat in Form eines eigenen Textabsatzes
body yes Inhaltsbereich eines HTML-Dokuments
br yes Erzwungener Zeilenumbruch an der betreffenden Stelle
button yes Schaltfläche in einem Formular
caption yes Tabellenüberschrift / -unterschrift
cite yes Zitat innerhalb eines Absatzes
code yes Quelltext-Auszug innerhalb eines Absatzes
col yes Tabellenspalte
colgroup yes Gruppe von Tabellenspalten
dd yes Definition, Beschreibung, detaillierter Inhalt zu einem dt-Element
del yes gelöschter Inhalt (Änderungsmarkierung)
dfn yes Definition innerhalb eines Absatzes
div yes Allgemeiner Bereich
dl yes Liste mit dt- und dd-Elementen, für Definitionslisten, Dialoge usw.
dt yes Zu definierender Ausdruck in einer Definitionsliste, Sprechername in einem Dialog usw.
em yes emphatischer Ausdruck innerhalb eines Absatzes (z.B. überspitzte Formulierung)
fieldset yes Gruppe von Elementen in einem Formular, Reiter in einem Dialog
form yes Formular, Dialog
h1-h6 yes Überschrift 1. bis 6. Ordnung
head yes Kopfbereich eines HTML-Dokuments
hr yes Trennlinie
html yes Dokumentelement von HTML-Dokumenten, schließt das gesamte Dokument ein
i yes spezieller Ausdruck im Text, z.B. Fachterminus oder Idiom (üblicherweise kursiv dargestellt)
img yes Referenzierte und eingebettete Grafik
input yes Formularelement, verwendbar für diverse Typen wie Eingabefelder, Schaltflächen, Auswahlfelder usw.
ins yes Eingefügter Inhalt (Änderungsmarkierung)
kpd yes Darstellung eines über Tastatur einzugebenden Textes im Text
label yes Beschriftungstext eines Formularfeldes
legend yes Überschrift für eine fieldset-Gruppe von Elementen in einem Formular
li yes Listenpunkt in einer ul-, ol- oder menu-Liste
link yes Logische Beziehungen der aktuellen Webseite zu anderen Webseiten
map yes Bereich für verweis-sensitive Grafiken
meta yes Meta-Angabe für nicht im Inhalt sichtbare Dokumentinformationen oder für Anweisungen an den Webserver
noscript yes Bereich für Inhalte, die angezeigt werden, wenn JavaScript im Browser deaktiviert oder nicht verfügbar ist
object yes Eingebettetes Objekt, geeignet für spezielle Multimedia-Objekte, Java-Applets usw.
ol yes Geordnete (z.B. nummerierte) Liste
optgroup yes Gruppe von Auswahlmöglichkeiten in einer Auswahlliste
option yes Auswahlmöglichkeit in einer Auswahlliste
p yes Paragraph
param yes Parameter eines Objekts bei object-Element
pre no Präformatierter Text (alle Whitespace-Zeichen werden im Browser so dargestellt wie im Text enthalten)
q yes Wörtliche Rede in einem Text
samp yes Beispieltext in einem dokumentierenden Text
script yes Scriptbereich für JavaScript und andere Scriptsprachen
select yes Auswahlliste in einem Formular
small no kleiner dargestellter Text als Normalschrift
span yes allgemeiner Bereich innerhalb eines Absatzes
strong yes allgemeiner Bereich innerhalb eines Absatzes
style yes Bereich für Stylesheet-Definitionen, z.B. in CSS
sub yes Tiefgestellter Text
sup yes Hochgestellter Text
table yes Tabelle
tbody yes Tabellenkörper mit den eigentlichen Daten
td yes Tabellenzelle
textarea yes Mehrzeiliger Eingabebereich in einem Formular
tfood yes Fußbereich in einer Tabelle, z.B. für summarische Daten
th yes Tabellenkopfzelle
thead yes Kopfbereich einer Tabelle
title yes Titel eines HTML-Dokuments
tr yes Tabellenzeile
tt yes »Fernschreiber«-Text
ul yes Unsortierte Liste (Aufzählungsliste)
var yes Variabler Name in einem dokumentierenden Text
